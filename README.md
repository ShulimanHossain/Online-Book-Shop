# Online Book Shop Management System

An Online Book Shop Web Application built with **Flask, MySQL, HTML, and CSS**, designed as a **database project**.  
It allows users to browse books, manage a shopping cart, place orders, and write reviews, while administrators can manage inventory, revenue, and users.

---

## Features


### Admin Features:

1. **Book Management**
    - Add, delete, or update books.
    - Add new authors or link books to existing authors.

2. **Stock & Pricing**
    - Manage book stocks
    - Update book prices

3. **Sales & Revenue Reporting**
    - View monthly revenue reports.
    - Track and analyze order summaries.

4. **User & Order Management**
    - View detailed user order history.
    - Manage all placed orders.

5. **Secure Login**
    - Role-based login system with sessions.
    - Admin password can be updated.

### Customer (Member) Features:

1. **Registration & Login**
    - Secure member login and registration.

2. **Book Catalog & Search**
    - Browse all books.
    - Search by book name and author name.

3. **Shopping Cart & Orders**
    - Add books to cart and update quantities.
    - Place orders with auto stock updates.
    - View order history.

4. **Book Reviews & Ratings**
    - Leave reviews and ratings for purchased books.
    - See other usersâ€™ reviews for books.

## Files 

- **app.py** : Main Flask application file.
- **config.py** : Database connection and configuration settings.
- **book.sql** :  SQL file containing the database schema 
- **templates** : HTML templates for frontend pages.
